# Digital-Literacy-VITyarthi-Project
Digital literacy is the ability to confidently find, evaluate, create, and communicate information using technology safely and responsibly. It combines technical skills (using devices/apps) with critical thinking to navigate digital environments, manage online privacy, and collaborate effectively in a 21st-century, technology-driven society.
